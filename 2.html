01상품 설명 자동화

스마트스토어, 쿠팡, 11번가 등 온라인 쇼핑몰에 올리는 상품 상세설명을 클로드가 대신 쓴다. 상품 정보만 넣으면 된다.

단품 상품 설명 생성

클로드코드에 입력
스마트스토어 상품 상세설명을 작성해줘.

상품명: [예: 여름 린넨 반팔 티셔츠]
소재: [예: 린넨 55% 면 45%]
특징: [예: 통기성 좋음, 세탁기 가능, S~XXL 사이즈]
가격: [예: 29,900원]
타겟: [예: 20~40대 직장인]
시즌: [예: 여름]

조건: 400자 내외, 이모지 포함, 사이즈 표 포함, 구매 유도 문구로 마무리. 파일명 product_desc.txt로 저장해줘.

상품 여러 개 한꺼번에 생성

엑셀에 상품 목록이 있으면 한 번에 전부 처리할 수 있다.

클로드코드에 입력
products.xlsx 파일을 열어줘. A열에 상품명, B열에 소재, C열에 특징이 있어.
각 행의 상품마다 스마트스토어 상세설명 300자를 D열에 채워줘.
완료되면 products_complete.xlsx로 저장해줘.
💡 상품 수가 많을수록 효과가 크다

10개 상품이면 기존에 2~3시간 걸리던 작업이 5분으로 줄어든다. 상품 목록 엑셀을 미리 만들어 오면 수업 중에 바로 적용해볼 수 있다.

11번가 · 쿠팡 스타일로 변환

클로드코드에 입력
방금 만든 product_desc.txt 내용을 쿠팡 스타일로 다시 써줘. 쿠팡은 짧고 직관적인 문장, 혜택 중심, 로켓배송 언급 포함. 쿠팡용은 coupang_desc.txt로 따로 저장해줘.
02SNS · 마케팅 문구 자동화

인스타그램 캡션 자동 생성

클로드코드에 입력
인스타그램 상품 홍보 캡션을 5가지 버전으로 만들어줘.

상품: [예: 린넨 티셔츠 신상 입고]
분위기: [예: 감성적이고 따뜻한 톤]
포함 내용: 가격(29,900원), 링크 유도("프로필 링크 클릭"), 해시태그 10개
각 버전은 150자 내외로 작성해줘.

고객 문의 답변 자동화

자주 받는 질문 유형을 등록해두면 클로드가 일관된 톤으로 답변 초안을 만들어준다.

배송 문의
배송 예정일 답변
주문일, 택배사, 예상 도착일을 넣으면 친절한 답변 초안 완성
⏱ 30초
교환·반품
교환·반품 안내
사유와 규정을 입력하면 정중하고 명확한 안내 문자 완성
⏱ 30초
상품 문의
사이즈·소재 안내
상품 스펙을 기반으로 고객 질문에 맞는 답변 자동 생성
⏱ 30초
클로드코드에 입력 — 고객 문의 답변
아래 고객 문의에 정중하고 친절한 답변을 작성해줘.

고객 문의: "주문한 지 3일 됐는데 배송이 안 왔어요. 언제 오나요?"
상황: 주문일 [날짜], 출고일 [날짜], 택배사 CJ대한통운, 송장번호 [번호]

톤: 정중하고 친절하게, 불편 드린 점 사과, 추적 방법 안내 포함. 200자 이내로 작성해줘.
03보고서 · 문서 자동화

주간 업무 보고서 자동 완성

클로드코드에 입력
이번 주 업무 내용으로 주간 보고서를 작성해줘.

완료 업무: [예: 신상품 20종 등록, 고객문의 31건 처리, 기획전 배너 제작]
진행 중: [예: 다음달 시즌오프 기획전 준비 60%]
이슈: [예: 특정 상품 재고 부족 — 발주 완료]
다음 주 계획: [예: 재고 실사, 신상품 촬영, 월말 정산]

형식: 표 + 항목별 요약. weekly_report_[날짜].docx로 저장해줘.

자주 쓰는 프롬프트 저장하기

매번 프롬프트를 다시 쓰지 않아도 된다. CLAUDE.md 파일에 자주 쓰는 지시를 저장해두면 클로드가 자동으로 참고한다.

  1. 작업 폴더 안에 CLAUDE.md 파일을 새로 만든다
  2. 아래 예시처럼 반복 지시 사항을 적는다
  3. 저장 후 클로드코드를 다시 실행하면 자동 적용된다
CLAUDE.md 예시
# 내 작업 규칙

- 모든 답변은 한국어로 작성
- 상품 설명은 항상 300~400자, 이모지 2~3개 포함
- 파일 저장 시 날짜를 파일명 끝에 붙임 (예: report_20260601.docx)
- 고객 답변 톤: 정중하고 친절하게, 존댓말 사용
- 엑셀 정리 후 항상 원본은 backup_ 접두어로 따로 저장
ℹ️ CLAUDE.md는 나만의 AI 비서 설명서

한 번 잘 만들어두면 매번 같은 조건을 반복 입력할 필요가 없다. 업무가 바뀌면 파일만 수정하면 된다.

04나만의 자동화 루틴 설계

지금까지 배운 기술을 조합해서 본인 업무에 맞는 자동화 흐름을 설계한다.

업무별 추천 자동화 조합

직업/업무추천 자동화예상 절약 시간
스마트스토어 셀러상품 설명 대량 생성 + 고객 문의 답변 템플릿하루 1~2시간
의류 브랜드 운영SNS 캡션 5종 세트 + 인스타 해시태그 생성주 3~4시간
직장인 (보고서)주간 보고서 자동 완성 + 회의 안건 정리주 2~3시간
프리랜서견적서 초안 + 이메일 회신 문구건당 30분
소상공인이벤트 문자 문구 + 메뉴 소개 문구주 2시간

실습 — 본인 루틴 완성하기

아래 질문에 답하면서 본인만의 자동화 루틴을 설계해본다.

📝 루틴 설계 질문지

1. 매일 또는 매주 반드시 하는 반복 업무는 무엇인가?
2. 그 중에서 가장 시간이 많이 걸리는 것은?
3. 결과물의 형태는? (파일, 문자, 게시글, 표…)
4. 조건이나 규칙이 있다면? (글자 수, 톤, 포함 요소…)
5. 저장은 어디에? (폴더, 파일명 규칙…)

본인 루틴 프롬프트 — 직접 작성해보기
[업무명]을 자동화해줘.

입력 정보: [필요한 정보 목록]
결과물 형태: [파일 종류, 분량, 형식]
조건: [톤, 스타일, 포함해야 할 요소]
저장 위치: [파일명 규칙]
💡 CLAUDE.md에 바로 저장해두자

오늘 만든 루틴 프롬프트를 CLAUDE.md에 추가해두면 다음번에 폴더만 열어도 바로 쓸 수 있다.

—트러블슈팅

결과물이 너무 길거나 짧다

글자 수를 명시한다

프롬프트에 "300자 이내", "5줄 이하", "A4 1페이지 분량"처럼 구체적인 분량을 지정하면 원하는 길이로 나온다.

엑셀 파일을 읽지 못한다

파일이 같은 폴더에 있어야 한다

클로드코드를 실행한 폴더 안에 엑셀 파일이 있어야 한다. VS Code에서 연 폴더 = 클로드코드의 작업 폴더다.

한 번에 너무 많이 시키면 중간에 멈춘다

작업을 쪼개서 요청한다

상품 100개를 한 번에 요청하는 것보다 20개씩 5번 나눠서 요청하는 것이 더 안정적이다. 큰 작업은 단계별로 나눈다.

파일을 저장했는데 폴더에 안 보인다

VS Code 탐색기를 새로고침한다

VS Code 왼쪽 탐색기에서 폴더 이름 오른쪽 클릭 → 새로고침을 누르면 새로 생성된 파일이 나타난다.
