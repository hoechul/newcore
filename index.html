강좌 소개

AI가 일을 대신하는 시대,
먼저 배우는 사람이 앞서갑니다

스마트스토어 상품등록, 엑셀 정리, 보고서 작성처럼 매일 반복되는 일들을 클로드 코드가 대신 처리합니다. 동시에 내 브랜드나 사업을 소개하는 랜딩페이지를 코딩 없이 직접 제작합니다.

소상공인, 직장인, 부업을 준비하는 누구에게나 바로 쓸 수 있는 실용적인 AI 활용 강좌입니다. 이론보다 실습, 개념보다 결과물 중심으로 진행합니다.

자주 묻는 질문

궁금하신 점이 있으신가요?

수강 전 가장 많이 물어보시는 질문들을 모았습니다.

Q1
코딩을 전혀 모르는데 따라갈 수 있을까요?
▾
네, 전혀 문제 없습니다. 본 강좌는 코딩 경험 제로인 분을 기준으로 설계되었습니다. 클로드 코드는 평소 대화하듯 한국어로 지시하면 AI가 코드를 대신 작성해 줍니다. 수강생 여러분은 "어떤 결과를 원하는지" 말하는 것만 익히시면 됩니다.
✔ 프로그래밍 언어 학습 없음✔ 한국어 지시만으로 실습
Q2
노트북은 꼭 가져와야 하나요? 어떤 사양이어야 하나요?
▾
노트북은 필수 지참입니다. 실습 중심 강좌이기 때문에 직접 화면으로 따라 하셔야 효과가 있습니다.
  • 운영체제: Windows 10/11 또는 macOS 모두 가능
  • 램 8GB 이상, 저장공간 여유 10GB 이상 권장
  • 강의장 Wi-Fi 제공 (인터넷 연결 필수)
  • 태블릿·스마트폰은 실습 불가
Q3
클로드 유료 구독이 필요한가요? 추가 비용이 있나요?
▾
강좌 중 실습은 Claude.ai Pro 구독(월 약 20달러)을 기준으로 진행합니다. 수강 전 미리 가입해 오시거나, 1회차 강의 시작 전 강사 안내에 따라 현장에서 가입하셔도 됩니다.
수강료 50만원 외 구독료 별도무료 플랜은 실습 일부 제한
Q4
3회 강좌가 끝나면 실제로 무엇을 만들 수 있게 되나요?
▾
3회 수료 후 직접 보유하게 되는 결과물입니다.
  • 내 업무 맞춤 자동화 프롬프트 모음
  • 상품 설명·SNS 문구·보고서 자동 생성 템플릿
  • 실제 인터넷에 공개된 나만의 랜딩페이지 1개
강의 중 완성한 결과물은 바로 현업에 적용 가능합니다.
Q5
어떤 종류의 반복 업무를 자동화할 수 있나요?
▾
강좌에서 다루는 자동화 업무 예시입니다.
  • 스마트스토어·11번가 상품명 및 상세설명 자동 생성
  • 엑셀 데이터 정리·분류·요약 자동화
  • 인스타그램·블로그 홍보 문구 대량 생성
  • 고객 문의 답변 초안 자동 작성
  • 주간·월간 업무 보고서 템플릿 자동 완성
Q6
만들어지는 랜딩페이지는 어디에 올릴 수 있나요?
▾
3회차 강의에서 Vercel을 활용한 무료 배포까지 실습합니다. 별도 서버 비용 없이 내 주소로 바로 공개됩니다.
  • 무료 호스팅 — 월 비용 0원
  • 스마트폰에서도 잘 보이는 반응형 디자인
  • 강의 후에도 본인이 직접 수정·업데이트 가능
Q7
1회차만 들어도 되나요? 중간에 빠지면 어떻게 되나요?
▾
강좌는 3회 전체가 연결된 과정으로 1회차 단독 수강은 불가합니다. 부득이하게 한 회를 빠지신 경우, 보충 자료를 통해 따라오실 수 있도록 지원해 드립니다. 다만 실습 특성상 직접 참여를 강력히 권장합니다.
Q8
환불은 언제까지 가능한가요?
▾
강남뉴코아 문화센터 공통 환불 규정을 따릅니다.
  • 개강 7일 전까지: 전액 환불
  • 1회차 수강 후: 수강료의 2/3 환불
  • 2회차 수강 후: 환불 불가
정확한 규정은 문화센터 공식 안내 기준으로 확인 필요
Q9
강의 내용이 금방 구식이 되지 않을까요?
▾
AI 도구 자체는 빠르게 업데이트되지만, 이 강좌에서 배우는 핵심은 "AI에게 어떻게 일을 시키는가"라는 사고방식입니다. 특정 버튼 위치가 바뀌어도 대화로 지시하는 방법은 변하지 않습니다. 도구가 바뀌어도 통하는 AI 활용 근육을 키우는 것이 목표입니다.
Q10
강좌가 끝난 후에도 질문하거나 도움을 받을 수 있나요?
▾
수료 후 수강생 전용 오픈 카카오톡 채널을 통해 사후 지원을 제공합니다.
  • 실습 중 생긴 오류·문제 질문 답변
  • 업무 적용 시 막히는 부분 도움
  • Claude 업데이트 주요 변경 사항 공유
✔ 수료생 커뮤니티 영구 참여 가능
