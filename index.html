왜 Claude Code인가

"나는 코딩을 전혀 몰라서요…"
그 말이 이 강좌의 시작입니다

기존 코딩 강좌와 전혀 다릅니다. Claude Code는 AI에게 한국어로 지시하면 모든 코드를 대신 써줍니다.

기존 방식 vs Claude Code 방식
❌ 기존 코딩 강좌
HTML·CSS·JS 문법 암기
영어 에러 메시지 해석
수백 줄 코드 직접 작성
6개월~1년 학습 필요
→
✅ Claude Code 방식
한국어로 "이렇게 해줘"만
AI가 코드 100% 작성
오류도 AI가 스스로 수정
오늘 배우고 내일 바로 사용

Q1
코딩을 한 번도 해본 적 없어도 따라갈 수 있나요?
▾
네, 완전히 가능합니다. 이 강좌는 코딩을 한 줄도 써본 적 없는 분을 기준으로 설계했습니다. Claude Code는 한국어로 "이렇게 만들어줘"라고 하면 AI가 모든 코드를 대신 작성합니다.
✔ 코딩 경험 제로도 OK✔ 영어 몰라도 OK✔ 한국어 대화만으로 완성
Q2
커리큘럼 순서가 셋팅 → 랜딩페이지 → 자동화인 이유가 있나요?
▾
1주차에 GitHub와 Vercel을 미리 세팅해 두면 2주차 랜딩페이지 완성 즉시 배포가 가능합니다. 매 주차마다 완성된 결과물을 손에 쥐고 돌아가도록 설계한 순서입니다.
Q3
노트북 사양이 어느 정도여야 하나요?
▾
  • 운영체제: Windows 10/11 또는 macOS 모두 가능
  • 램 8GB 이상, 저장공간 여유 10GB 이상 권장
  • 태블릿·스마트폰은 실습 불가 — 노트북 필수
Q4
Claude 유료 구독이 꼭 필요한가요?
▾
Claude.ai Pro 구독(월 약 $20)이 필요합니다. 무료 플랜은 실습 도중 한도에 걸릴 수 있습니다.
수강료 50만원 외 별도무료 플랜은 실습 제한
Q5
3회 수료 후 실제로 무엇을 갖게 되나요?
▾
  • 실제 URL이 있는 나만의 랜딩페이지 1개
  • 내 업무에 맞춘 자동화 프롬프트 모음(CLAUDE.md)
  • 상품 설명·SNS 문구·보고서 자동 생성 템플릿
  • 수료생 전용 카카오톡 채널 영구 참여

▾
GitHub + Vercel 무료 플랜으로 월 운영 비용 0원입니다. 독자 도메인 원할 때만 연 1~3만원.
